# pkgpick

[English](README.md) | [Русский](README_RU.md)

TUI-менеджер пакетов для Arch Linux на базе fzf.

pkgpick позволяет искать, устанавливать, обновлять, удалять и смотреть инфо о пакетах из официальных репозиториев, AUR, Flatpak, npm, pip, cargo, go и pipx — всё из одного интерфейса.

## Возможности

- Поиск и установка пакетов из официальных репозиториев, AUR и Flatpak
- Управление установленными пакетами поотдельно либо всеми сразу в комбинированном списке
- Кнопка полного обновления всего источника (например, «oбновить всё из AUR»)
- Опциональная поддержка npm, pip, cargo, go и pipx (включается в Settings, или разово через `--full`)
- Встроенное меню Cleanup: кеши пакетных менеджеров, orphan-пакеты, кеш AUR-хелпера, неиспользуемые flatpak-runtime'ы
- Живая сортировка/фильтрация списков установленных пакетов: по имени, размеру, дате установки, explicit/dependency
- Vim-навигация (`h`/`j`/`k`/`l`) в любом меню без поиска, наряду с обычными стрелками
- Enter на вопросе да/нет по умолчанию означает «да»
- Русский и английский интерфейс, переключается в любой момент из Settings
- Лёгкий, почти без зависимостей, полностью терминальный

## Установка

```bash
yay -S pkgpick
```

или

```bash
paru -S pkgpick
```

## Использование

```bash
pkgpick
```

Выберите пакет и нужное действие.

## Требования

- Arch Linux
- bash
- fzf
- pacman

Опционально:

- yay или paru — поддержка AUR
- flatpak — Flatpak-приложения/runtime'ы
- npm, pip, cargo, go, pipx — появляются в меню источника, только если установлены

## Языки интерфейса

pkgpick поддерживает:

- Русский
- English

При первом запуске предлагается выбрать язык; выбор сохраняется и его можно поменять в любой момент из меню Settings.
