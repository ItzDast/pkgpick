# pkgpick

[English](README.md) | [Русский](README_RU.md)

Простой TUI-выборщик пакетов для Arch Linux.

pkgpick — это интерфейс на базе fzf для поиска и установки пакетов из официальных репозиториев, AUR и уже установленных пакетов.

## Возможности

- Поиск пакетов через fzf
- Установка пакетов из официальных репозиториев
- Установка пакетов из AUR
- Работа с установленными пакетами
- Поддержка русского и английского интерфейса
- Лёгкий и полностью терминальный интерфейс

## Установка

```bash
yay -S pkgpick
```

или

```bash
paru -S pkgpick
```

## Использование

```bash
pkgpick
```

Выберите пакет и нужное действие.

## Требования

- Arch Linux
- bash
- fzf
- pacman

Опционально:

- yay
- paru

## Языки интерфейса

pkgpick поддерживает:

- Русский
- English

Язык интерфейса автоматически определяется по локали системы.
